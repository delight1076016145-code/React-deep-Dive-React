실행컨텍스트 스택이 있고

전역 컨텍스트 객체가 있음.

자바스크립트 함수는 객체이다.

요즘 추세는 애로우 함수 형태로 선언 해서 변수에 할당하는 방식을 선호

var 사용 지양 let, const 사용하길 권장

파라미터 -> 선언적함수 -> var 순으로 호이스팅 된다.

전역 실행 컨텍스트 -> 각 함수 실행 컨텍스트 스택으로 쌓이고 모두 실행 된 후 -> 전역 실행 컨텍스트만 남는다.

스콥체인 - 자기 자신을 호출했던 영역에 있던 변수를 찾다가 없으면 상위에서 동일한 변수가 있는지를 찾는것.
영역에 없으면 함수가 호출된 곳에서 변수가 정의 되었는지 찾고 없으면 최상단 전역 영역에 함수 가 있는지 찾으면 됨.

let, const는 스콥을 구분하는 기능을 추가된 변수 선언

호이스팅은 if문의 영향을 받지 않고 var가 있는지만 찾는다.

정적 타입은 변수 타입을 미리 선언 하는것.
동적 타입은 아무 타입이나 들어 갈 수 있는것
